bool Top(const int auth=0, const int work=0, const int sect=0)

Moves to the top (beginning) of author, work, or section indicated by the three 0-based indexes. E.g., to go to the beginning of work 2, section 5: 

	tf->Top(0,1,4)
You are now ready to start reading line from the work with Next() or SearchNext().

int Count(const int auth = -1, const int work = -1)

Returns the count of authors, works, or sections 

	int a = tf.Count();    // number of authors in the file
	int b = tf.Count(0);   // number of works by author 0
	int c = tf.Count(0,3); // number of sections in work 3 of author 0
